## 🚀 Quick Start

### Option 1: Blueprint Installation (Recommended)
```bash
blueprint install feathereggs
# Done! ✨
```

### Option 2: Manual Installation

## 1. Extract the Files
Unzip the FeatherEggs folder into your Pterodactyl directory.

---

## 2. Add the Assets
Open:
```
resources/views/layouts/scripts.blade.php
```

Scroll to the end of the file and paste the following:

```html
<script src="/extensions/feathereggs/inject-admin-nest-list.js"></script>
<script src="/extensions/feathereggs/inject-admin-nest-import-modal.js"></script>
<script src="/extensions/feathereggs/inject-admin-sidebar.js"></script>
<script src="/extensions/feathereggs/inject-admin-nest-page.js"></script>
```

✅ This enables the panel to inject everything FeatherEggs needs.

---

## 3. Add the Admin Routes
Next, open:
```
routes/admin.php
```

Go to the bottom of the file and add:

```php
Route::group(['prefix' => 'extensions'], function () {
    Route::get('/feathereggs', [Admin\Extensions\FeatherEggsController::class, 'index'])->name('admin.extensions.feathereggs');
    Route::post('/feathereggs', [Admin\Extensions\FeatherEggsController::class, 'post'])->name('admin.extensions.feathereggs.post');
});
```
