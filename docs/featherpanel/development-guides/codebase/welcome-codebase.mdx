# Codebase

Welcome to the page where you can learn more about the codebase and how FeatherPanel is built!


## ðŸ”§ Development Tools

### Backend Development
- **PHP 8.1+**: Modern PHP features, type hints, and attributes.
- **MythicalFramework**: MVC framework with Eloquent ORM.
- **Composer**: PHP dependency management.
- **PHP CS Fixer**: Code style and formatting.

### Frontend Development
- **Vue 3**: Composition API and TypeScript support.
- **Vite**: Fast development server and build tool.
- **Pinia**: State management for Vue.
- **Tailwind CSS**: Utility-first CSS framework.
- **ESLint & Prettier**: Code quality and formatting.

### DevOps & Deployment
- **Docker**: Containerization for both development and production.
- **GitHub Actions**: CI/CD pipelines for backend, frontend, and Docker builds.
- **Nginx**: Web server for serving the application.
- **MySQL**: Database for persistent storage.
