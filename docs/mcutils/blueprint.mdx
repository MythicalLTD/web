# Blueprint Installation


## Prerequisites

Before installing MCUtils, make sure you have Blueprint Framework installed on your system. If you haven't installed Blueprint yet, please follow the [Blueprint installation guide](https://blueprint.zip/docs/?page=getting-started/Installation).

## Installation
To install MCUtils using Blueprint, simply run the following command:

```bash
blueprint -i mcutils.blueprint
```
