    Name: `settings.updateSetting`

    Description: `Update a setting in the database.`

    ```php title='Example'
    $eventHandler->on('settings.updateSetting', function ($category, string $name, string $value, bool $updateCache) {
        //Do whatever you think is good to do idk ;)!
    });
    ```

    Arguments
     * @param string $category The name of the category
     * @param string $name The name of the setting
     * @param string $value The value you want to replace with!
     * @param bool $updateCache Update the cache after updating the setting
