# BackupEvent

Name: `backup.`

Description: `Backup events are events that are fired when a backup is created or restored.`

Since: `1.0.0`

## Events

| Event Name                 | Description                                                       | Since |
|----------------------------|-------------------------------------------------------------------|-------|
| [onBackupCreate][1]         | This event is getting fired when a backup is created!            | 1.0.0 |
| [onBackupRestore][2]        | This event is getting fired when a backup is restored!           | 1.0.0 |
| [onBackupRemove][3]         | This event is getting fired when a backup is removed!            | 1.0.0 |

[1]: #onBackupCreate
[2]: #onBackupRestore
[3]: #onBackupRemove

### `onBackupCreate`

FullName: `backup.onBackupCreate`

Description: `This event is getting fired when a backup is getting created!`

Since: `1.0.0`

Args:
- `$path` - The path to the backup file

Run:
- After the backup is created this event is fired!

### `onBackupRestore`

FullName: `backup.onBackupRestore`

Description: `This event is getting fired when a backup is getting restored!`

Since: `1.0.0`

Args:
- `$path` - The path to the old backup file

Run:
- After the backup is restored this event is fired!

### `onBackupRemove`

FullName: `backup.onBackupRemove`

Description: `This event is getting fired when a backup is getting removed!`

Since: `1.0.0`

Args:
- `$path` - The path to the old backup file


Run:
- After the backup is removed this event is fired!
