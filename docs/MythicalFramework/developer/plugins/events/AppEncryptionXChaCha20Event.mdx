# AppEncryptionXChaCha20Event

Name: `app.encryption.xchacha20.`

Description: `App encryption xchacha20 events are events that are fired when the app is using the xchacha20 encryption.`

Since: `1.0.0`

## Events

| Event Name                 | Description                                                            | Since |
|----------------------------|------------------------------------------------------------------------|-------|
| [onAppEncryptionXChaCha20Encrypt][1] | This event is getting fired when the app is encrypting data with xchacha20! | 1.0.0 |
| [onAppEncryptionXChaCha20Decrypt][2] | This event is getting fired when the app is decrypting data with xchacha20! | 1.0.0 |
| [onAppEncryptionXChaCha20Key][3]     | This event is getting fired when the app is generating a new key for xchacha20! | 1.0.0 |

[1]: #onappencryptionxchacha20encrypt
[2]: #onappencryptionxchacha20decrypt
[3]: #onappencryptionxchacha20key


### `onAppEncryptionXChaCha20Encrypt`

FullName: `app.encryption.xchacha20.onAppEncryptionXChaCha20Encrypt`

Description: `This event is getting fired when the app is encrypting data with xchacha20!`

Since: `1.0.0`

Args:
- `$data` - The data that is getting encrypted

Run:
- After the data is encrypted this event is fired!


### `onAppEncryptionXChaCha20Decrypt`

FullName: `app.encryption.xchacha20.onAppEncryptionXChaCha20Decrypt`

Description: `This event is getting fired when the app is decrypting data with xchacha20!`

Since: `1.0.0`

Args:
- `$data` - The data that is getting decrypted

Run:
- After the data is decrypted this event is fired!


### `onAppEncryptionXChaCha20Key`

FullName: `app.encryption.xchacha20.onAppEncryptionXChaCha20Key`

Description: `This event is getting fired when the app is generating a new key for xchacha20!`

Since: `1.0.0`

Args:
- `$key` - The key that is getting generated

Run:
- After the key is generated this event is fired!
